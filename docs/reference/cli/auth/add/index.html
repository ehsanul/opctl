opctl auth add

opctl auth add RESOURCES [ -u=<username> ] [ -p=<password> ]

Add auth for an OCI image registry.

#Arguments

#RESOURCES

Resources this auth applies to in the form of a host or host/path.

#Options

#-u or --username

Username

#-p or --password

Password

#Global Options

see global options

#Examples

#docker.io

opctl auth add docker.io -u <username> -p <password>
